# SQL Fraud Detection Patterns

## Minor Project

Fraud Detection using SQL queries based on real-world banking fraud scenarios.

### Concepts Used
- WHERE
- GROUP BY
- HAVING
- CASE WHEN
- CTE (WITH)
- ROW_NUMBER()
- LAG()
- DATEDIFF()
- TIMESTAMPDIFF()
- Window Functions
- JOIN
- Aggregate Functions

### Fraud Patterns Implemented

- Velocity Fraud
- Round Amount Clustering
- Card Testing
- Failed-Then-Succeeded
- Odd Hour Concentration
- Mule Accounts
- Refund Abuse
- Merchant Collusion
- Structuring (Smurfing)
- Dormant-Then-Active
- Velocity Spike
- Geographic Impossibility

### Technologies

- MySQL 9.x
- MySQL Workbench
